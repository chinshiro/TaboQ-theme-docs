# Theme Settings Overview

Navigate to **Customize → Theme Settings** to configure:

## 🎨 Colors & Typography
- Set primary/secondary colors
- Choose fonts for headings, body, buttons

## 🖼️ Media & Logo
- Upload logo (SVG, PNG, JPG)
- Set favicon
- Configure image ratio and lazy loading

## 🧭 Header & Navigation
- Choose layout: Classic / Mega Menu / Drawer
- Enable search, cart, language/currency picker
