# Frequently Asked Questions

<details>
<summary>How do I change the homepage layout?</summary>
Go to **Customize → Homepage** and use Sections to drag-and-drop blocks like Slideshow, Featured Products, Rich Text, etc.
</details>

<details>
<summary>Why isn’t my slideshow showing images?</summary>
Make sure you’ve:
- Added at least one slide
- Uploaded images (recommended: 1800x800px)
- Published the theme
</details>

<details>
<summary>How do I enable cart drawer?</summary>
Go to **Theme Settings → Cart → Cart Type → Drawer**.
</details>
