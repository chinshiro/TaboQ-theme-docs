# Installation Guide

## 1. Download Your Theme

After purchase, download the `.zip` file from your Shopify Theme Store account or email.

## 2. Upload to Shopify

1. Go to **Shopify Admin → Online Store → Themes**
2. Click **“Add theme” → “Upload zip file”**
3. Select the downloaded `.zip` file
4. Wait for upload to complete → Click **“Actions → Publish”**

:::caution
Always **duplicate your live theme** before making any code edits!
:::
